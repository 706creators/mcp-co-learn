## 共学前置

参与这个共学需要知道

1. 如何友好交流
2. 如何使用 github
3. Mac/Linux 开发环境
4. 至少具备一门编程语言基础
   
## 共学模块

### 模块一

1. W1-1：MCP 基本介绍
   介绍 AI 开发的基本知识及 MCP 协议的核心概念和应用场景。
3. W1-2：Client 调用
   掌握在 Client 端调用 MCP 的原理和实现方法，理解数据交互的流程。

### 模块二

1. W2-1：Server 调用
   学习如何编写 MCP Server，使其能够与 Client 端进行有效的数据交互。
3. W2-2：案例分析
   通过实际操作，构建一个完整的 MCP Agent 实例，提升对 MCP 协议的实战应用能力。
